# 版本规划文档（单人开发版）

> 项目：今天刷牙了吗（Dentic）— 儿童刷牙引导微信小程序
> 当前阶段：已有 100+ 用户
> 规划原则：**先做最小有效功能，不做复杂系统，不做重工程化设计**
> 一句话策略：**先把"刷牙引导"做顺，再把"家庭协作"做轻，最后再补"晨晚场景信息"。**

## V1.1 — 主链路优化版

> 目标：让用户第一次进来就知道——这不是普通计时器，而是给孩子用的刷牙引导工具。

### 1. 首页优化

| 状态 | 模块 | 说明 |
|------|------|------|
| ✅ | 模块 A：今日状态 | 卡片化展示早晚刷状态，完成显示 ✓ + 时间，待完成灰色 |
| ✅ | 模块 B：开始刷牙 | 保持圆形脉冲按钮，系统自动判断早晚场景 |
| ✅ | 模块 C：一句价值说明 | 「科学引导 15 区域刷牙，每次只需 2.5 分钟」 |
| ✅ | 模块 D：连续打卡 | 连续天数 + 里程碑进度条 |

### 2. 刷牙流程优化

| 状态 | 功能 | 说明 |
|------|------|------|
| ✅ | 当前区域展示 | 牙齿图高亮当前区域 + StepIndicator 显示区域名 + 文字提示 |
| ✅ | 当前区域倒计时 | BrushTimer 圆环倒计时，显示剩余秒数 |
| ✅ | 流程进度 | StepIndicator 15 段进度条 + "N/15" 计数 |
| ✅ | 下一步提示 | 卡片底部显示「下一步：XX」，最后一步显示「最后一步！」 |
| ✅ | 切换反馈 | 震动+音效+语音+卡片入场动画（key 触发重新挂载） |

### 3. 完成页优化

| 状态 | 功能 | 说明 |
|------|------|------|
| ✅ | 完成祝贺 + 统计数据 | 随机鼓励语 + 用时/步数/连续天数 |
| ✅ | 里程碑消息 | 达成里程碑时显示特殊消息 |
| ✅ | 今日剩余任务提示 | 完成页显示「记得晚上再来刷一次哦」或「明天见！」 |
| ✅ | 下次回来提醒 | 根据早晚场景自动生成回来提示文案 |

### 4. 今日状态统一

| 状态 | 功能 | 说明 |
|------|------|------|
| ✅ | 今日早刷状态 | `getTodayDailyStatus()` 已追踪 morningDone |
| ✅ | 今日晚刷状态 | `getTodayDailyStatus()` 已追踪 eveningDone |
| ✅ | 今日任务完成（早+晚合并） | `isDailyComplete()` 辅助函数 |
| ✅ | 连续天数 | `getCurrentStreak()` 已实现 |
| ✅ | 首页/完成页/我的页面统一展示 | 完成页已使用 dailyStatus 展示回来提示 |
