# 设计系统：刷了吗 (Dentic)

## 1. 视觉主题与氛围

整体美学追求**温暖、克制、类纸质的有机感**。界面像一本精心排版的手帐——奶油色的底色如羊皮纸般柔和，赤陶色的点缀带来泥土般的温度，阴影轻如耳语，边框若隐若现。设计的核心理念是"少即是多"：没有浓烈的渐变、没有炫目的动效，一切都在安静中传递秩序和信任。

**关键特征**：
- **密度**：疏朗通透，大量留白，内容自然呼吸
- **质感**：类纸质的温柔触感，远离冰冷的数字工业感
- **情绪**：平和、鼓励、不施压——适合每天两次的日常仪式
- **风格**：编辑式排版（editorial typography），衬线与无衬线字体的和谐对话

### 透明度模式

通过透明度修饰符构建视觉层次，而非定义大量独立的灰色变体：
- **文字层次**：`content`(100%) → `content/50` → `content/40` → `content/25`
- **边框层次**：`content/[0.06]` → `content/[0.08]` → `content/[0.12]`
- **底色层次**：`content/[0.03]`（极淡底纹） → `content/[0.06]`（微弱底纹）

## 3. 字体规则

### 字体配对

- **Poppins**（几何无衬线）：所有 UI 控件、标题、标签、数字。使用 600（SemiBold）和 700（Bold）两个字重。传递现代感和清晰度。
- **Lora**（优雅衬线）：仅用于首页的编辑风格问候语。使用 400（Regular）和 500（Medium）字重。传递温度和文学气质。

这种"无衬线 UI + 衬线编辑"的配对，让界面在功能性和人文温度之间找到平衡。

Section 标题、日期标签等使用统一的"小号大写字母"模式：
- `text-label-sm` + `font-heading` + `font-semibold` + `tracking-[0.1em]` + `uppercase` + `text-content/50`

数字统一使用 `tabular-nums` 确保等宽对齐。

## 4. 间距体系

### 语义间距 Token

| Token | 值 | 用途 |
|-------|-----|------|
| `page-x` | 1.5rem (24px) | 页面水平内边距 |
| `page-gap` | 1.25rem (20px) | 页面级内容块之间的间距 |
| `section-gap` | 2rem (32px) | 板块之间的分隔间距 |
| `bottom-safe` | 7rem (112px) | 底部导航安全区（底部内边距） |

### 组件级间距

| 场景 | 值 | 说明 |
|------|-----|------|
| 卡片内边距 | `p-5` (20px) | CardContent 统一内边距 |
| 列表项内边距 | `px-4 py-3.5` (16px/14px) | ListItem 标准内边距 |
| 列表项最小高度 | `min-h-[3.75rem]` (60px) | 确保触控友好 |
| 状态行间距 | `gap-3` (12px) | 信息行之间的间距 |
| 按钮行间距 | `gap-2` (8px) | 并排按钮之间的间距 |

### 排版节奏

页面内容遵循"板块-区域-元素"三层间距节奏：
- **板块间**（Section 级）：`section-gap` (32px) — 不同功能区域之间的呼吸
- **区域间**（内容块级）：`page-gap` (20px) — 同一板块内的内容分组
- **元素间**（组件级）：`gap-2`~`gap-4` (8px-16px) — 紧密关联的元素

## 5. 组件样式

### 按钮 (Button)

药丸形主操作按钮使用圆角全圆（`rounded-full`），标准按钮使用柔和圆角（6px）。按压反馈通过微缩放（`scale-[0.98]`）和透明度变化（`opacity-90`）实现，过渡时长 200ms。

- **主按钮**：赤陶色填充底色 + 白色文字
- **次按钮**：纯白底色 + 近黑文字 + 极淡边框 (`content/[0.12]`)
- **轮廓按钮**：透明底色 + 近黑文字 + 淡边框 (`content/[0.2]`)
- **幽灵按钮**：全透明 + 次要灰文字

### 卡片与容器 (Card)

柔和圆角（6px `rounded-anthropic`），纯白底色，几乎不可感知的轻影（`0 1px 3px rgba(20,20,19,0.04)`）和极淡边框（`content/[0.08]`）。卡片不是"悬浮"的，而是"轻搁在纸面上"的感觉。

完成态卡片使用更大的圆角（12px `rounded-anthropic-lg`），传递庆祝和特殊感。

### 列表 (List)

容器与卡片共享圆角和边框风格。列表项之间使用极淡分隔线（`content/[0.06]`）分割。每行最小高度 60px，确保手指友好的触控区域。

### 徽章 (Badge)

极小的圆角（4px `rounded-anthropic-sm`），色彩变体通过对应功能色的淡底色 + 边框 + 文字色三层构建。标签使用最小号字体 + 宽字距。

### 开关 (Switch)

40x24px 尺寸，圆形轨道。开启态为赤陶色轨道 + 白色圆形拇指；关闭态为淡灰轨道。过渡动画 300ms。

## 6. 布局原则

### 页面壳模式

所有页面共享统一的容器结构：
- 外层：`theme-page app-scroll theme-day min-h-screen`
- 内层：`pb-bottom-safe px-page-x max-w-2xl mx-auto flex flex-col gap-page-gap`
- 顶部安全区通过动态计算（状态栏 + 导航栏 + 菜单按钮高度）设置 `paddingTop`

### 底部导航

固定在视口底部，半透明白色背景 + 模糊效果（`backdrop-blur-sm`）。底部内边距 32px 适配设备安全区。三个 Tab 使用 emoji 图标 + 极小标签文字，活跃态为赤陶色，非活跃态为 40% 透明度。

### 首页特殊布局

首页根据状态采用不同的布局策略：
- **Idle 态**：锁定视口高度，内容通过 flex column + `mt-auto` 将 CTA 按钮固定在底部
- **Active 态**：上部 42% 放牙齿可视化，下部弹性区放控制面板
- **Completed 态**：允许滚动，单张大卡片居中展示

## 7. 动画与过渡

### 入场动画

首页使用**交错式淡入上升**（`fade-up`）——内容块依次从下方 10% 位移处浮入，间隔 100ms，使用弹性缓动曲线（`cubic-bezier(0.16, 1, 0.3, 1)`）。这种"瀑布式入场"为静态内容增添了节奏感。

### 微交互

- **倒计时脉冲**：数字从 50% 缩放膨胀到 120% 再回落到 100%，0.8s，配合透明度变化
- **完成弹跳**：成功对勾图标缓慢呼吸式缩放（1→1.05→1），1s 循环
- **CTA 箭头**：水平方向 6px 弹跳，暗示"向前"
- **进度指示器脉冲**：当前步骤段的透明度在 1 和 0.7 之间柔和呼吸，2s 循环

### 无障碍

所有动画通过 `motion-reduce:animate-none` 支持减弱动画偏好。用户可在系统设置中关闭动画而不影响功能。

## 8. 圆角与阴影

### 圆角规则

| Token | 值 | 用途 |
|-------|-----|------|
| `anthropic` | 6px | 标准容器——卡片、按钮、输入框 |
| `anthropic-sm` | 4px | 小型元素——徽章、小按钮 |
| `anthropic-lg` | 12px | 特殊容器——完成态卡片 |
| `full` | 完全圆形 | 开关、药丸按钮、头像 |

### 阴影规则

阴影极度克制，几乎不可感知——目的是传递"微微浮起"而非"悬浮在空中"：

| Token | 值 | 用途 |
|-------|-----|------|
| `card` | `0 1px 3px rgba(20,20,19,0.04)` | 默认卡片——仅在底部投下一丝暗影 |
| `card-hover` | `0 4px 12px rgba(20,20,19,0.08)` | 悬浮态——轻微加深 |
| `card-lg` | `0 8px 24px rgba(20,20,19,0.06)` | 大卡片——更大扩散但仍然柔和 |
| `nav` | `0 -1px 0 rgba(20,20,19,0.06)` | 底部导航顶部——单像素线阴影 |
